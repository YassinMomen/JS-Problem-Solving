# JavaScript Problem Solving

This repository contains my solutions to around 100 JavaScript problems, primarily from Codewars.

My main background is in Competitive Programming using C++ (with over 1,600 problems solved). I created this track to bridge the gap between my algorithmic thinking in C++ and modern JavaScript development.

The focus here is on getting comfortable with ES6+ syntax, understanding the language's core behavior, and writing clean code using JavaScript's built-in array methods (map, filter, reduce) instead of traditional loops when appropriate.

This serves as my preparation phase before moving heavily into backend development with Node.js.
